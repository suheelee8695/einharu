## Image filename convention

`einharu_<category>_<product-slug>_<color>_<silhouette>_<view>.jpg`

Examples:
- `einharu_tops_bijo-shirt_charcoal_oversized_front.jpg`
- `einharu_pants_wide-leg-trousers_cream_drape_side.jpg`
- `einharu_dresses_vegan-leather-midi_black_matte_detail.jpg`

## Alt-text rules

- Describe what is visible first, then add one contextual phrase.
- Keep it natural and specific.
- Avoid repeating multiple keywords in one alt string.
- Use one language per page context.
